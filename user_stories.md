# User Stories for Smart Clinic Management System

## 1. Doctor User Stories
* **As a Doctor**, I want to view my daily appointment schedule so that I can manage my time effectively.
* **As a Doctor**, I want to record prescriptions for patients during their appointments so that they have a digital record of their treatment.
* **As a Doctor**, I want to set my available time slots so that patients can only book when I am free.

## 2. Patient User Stories
* **As a Patient**, I want to search for doctors by their specialty so that I can find the right consultant for my health issue.
* **As a Patient**, I want to book an appointment online so that I don't have to wait in a long queue.
* **As a Patient**, I want to view my previous prescriptions and medical history for future reference.

## 3. Admin User Stories
* **As an Admin**, I want to add or remove doctors from the system to keep the clinic's staff list updated.
* **As an Admin**, I want to generate reports on daily appointments to monitor the clinic's performance.
* **As an Admin**, I want to manage user accounts and ensure the system's security.
